## 🌌 Философия проекта

Интернет должен быть свободным, а инструменты для его защиты — удобными. 

KarinCore создана для того, чтобы навсегда избавить пользователей Linux от боли настройки сложных консольных утилит, написания бесконечных JSON-конфигов и ручного управления системными службами. Это мост между мощными низкоуровневыми протоколами обхода блокировок (Xray) и интуитивно понятным, эстетичным графическим интерфейсом.

Здесь нет тяжеловесного Electron и "песочниц", ломающих системную маршрутизацию. Только легковесное ядро на Rust, нативная интеграция с `systemd` и молниеносный UI.

## ✨ Ключевые возможности

*   **Многоуровневая маршрутизация (Режим Матрёшки):** Полноценная нативная интеграция протоколов OpenVPN и WireGuard. Заворачивайте ваш Xray трафик (VLESS/VMess и др.) бесшовно внутрь зашифрованного OVPN или WG туннеля.
*   **Нативная интеграция и изоляция сессий:** Ядро работает как фоновый системный демон (`karin-proxy-daemon.service`). Внедрен умный контроль MTU (MSS Clamping) и автоматическая зачистка туннелей перед новым подключением для защиты от утечек.
*   **Бесшовный UX:** Благодаря правилам `/etc/sudoers.d`, управление интерфейсами, маршрутизацией и Geo-базами происходит в один клик. Никаких раздражающих всплывающих окон с вводом пароля root.
*   **Интерактивный ассистент:** Познакомьтесь с Карин — вашей встроенной кибер-помощницей. Она следит за ядром, реагирует на ваши действия и уважает цифровое пространство, не перегружая его визуальным шумом.
*   **Умная маршрутизация и апдейты:** Встроенная поддержка режимов Direct, Proxy и Block с автоматической загрузкой баз GeoIP/GeoSite с нужными правами и тихой проверкой обновлений через GitHub API.
*   **Нулевая телеметрия и абсолютная приватность:** KarinCore работает исключительно локально. Никаких скрытых трекеров, сбора данных или аналитики. Ваши ключи шифрования, IP-адреса и профили маршрутизации никогда не покидают ваше устройство. Проект распространяется по открытой лицензии MIT.

## 🚀 Установка

KarinCore разработана с прицелом на максимальную совместимость с современными дистрибутивами Linux.



### Arch Linux (AUR)
Рекомендуемый способ установки для пользователей Arch-based систем (Manjaro, EndeavourOS и др.). Пакет автоматически соберет ядро, установит зависимости и настроит системные службы.

```bash
yay -S karincore-git
```

### Ubuntu / Debian / Linux Mint

В разделе [Releases](../../releases) всегда доступен свежий `.deb` пакет. Он автоматически настроит правила `sudoers` и `systemd` во время распаковки. Убедитесь, что в вашей системе установлены пакеты `openvpn` и `wireguard-tools`.

```bash
sudo dpkg -i KarinCore_1.2.1_amd64.deb
sudo apt install -f # если потребуются дополнительные зависимости
```



### Запуск и использование

В отличие от стандартных системных служб, демон KarinCore **не нужно** добавлять в автозагрузку (чтобы он не перехватывал трафик до того, как графический интерфейс сгенерирует рабочий конфиг).

**Не используйте** `systemctl enable`. Просто запустите KarinCore из меню приложений вашего десктопного окружения. Графический интерфейс сам запустит, настроит и безопасно остановит демона, когда это необходимо, используя заранее подготовленные правила sudo.



## 🛠 Архитектура (Для гиков)

Приложение разделено на два независимых бинарника:

    Backend (karin-proxy-daemon): Системный сервис, написанный на чистом Rust. Запускается от имени root, управляет TUN-интерфейсами, правилами маршрутизации и работает с сетью.

    Frontend (karincore): Легковесный GUI на Tauri. Работает в пространстве пользователя (user-space), общается с демоном через IPC/сокеты и безопасно перезапускает его через заранее настроенные правила sudoers.

Такое разделение прав позволяет сохранить систему в безопасности, не запуская весь графический стек с правами суперпользователя.



## 🗺️ Вектор развития (Roadmap)

Хотя KarinCore изначально создавалась для Linux, у свободного интернета нет привязки к ОС. 
**Сейчас в работе кроссплатформенная экспансия:** разработка нативных версий для Windows и macOS — моя следующая крупная цель. Карин планирует захватить и другие операционные системы в самом ближайшем будущем!
